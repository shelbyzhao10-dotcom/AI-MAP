# core elements
## take more neutral farming
用中立的框架语言不能有自己主观的情感色彩,防止AI出现谄媚现象
## Role prompt
角色扮演
## input fully context
越充足具体的上下文越能让AI了解我们的需求，从而获得更加契合我们需要的回答





# improve writing skills reduce slop
## Write  progressive outline
在写作前先让AI 生成一个大纲，之后在反复back and forth 的修改大纲，之后让AI照着大纲写。
## 2,Editing Piece by piece
基于大纲生成好文章后修改的时候一段一段的修改，避免返工。在修改或生成时可以让AI尝试不同风格的语气去写作例如，学术风，新闻报告风等
## 3，When optimizing,we can first create a multi-dimensional scoring table
在优化写作的内容时可以从多维度去生成一份评分表用客观的逾期让AI去评分(最好跨AI评分)

# 四，generate images 
1,learn how to describe a picture(we can let AI to help us to describe it that use accurate words )
当我们想去生成图片时，可先和AI聊想法，最后让AI用专业的术语（一些针对光影，角度等细节）来表达我们对这张图片的需求，在用它的描述去当作prompt去生图
